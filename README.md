## 🚀 Quick Start / 快速开始

```bash
# Clone the repository / 克隆仓库
git clone https://github.com/your-username/cloud-demo.git
cd cloud-demo

# Build the project / 构建项目
mvn clean install

# Start services / 启动服务
cd services/service-product
mvn spring-boot:run
```

## 🎯 Project Overview / 项目简介

A microservices e-commerce system demonstration project based on Spring Cloud Alibaba, showcasing core functionalities and best practices of microservices architecture.

基于 Spring Cloud Alibaba 的微服务电商系统演示项目，展示了微服务架构的核心功能和最佳实践。

---

## 💡 Key Features / 核心特性

- ✅ Service Registration & Discovery / 服务注册与发现
- ✅ Configuration Management / 配置管理
- ✅ API Gateway / API网关
- ✅ Service Invocation / 服务调用
- ✅ Circuit Breaking / 熔断降级
- ✅ Load Balancing / 负载均衡

---

## 📚 Tech Stack / 技术栈

- **Java 17**
- **Spring Boot 3.3.4**
- **Spring Cloud 2023.0.3**
- **Spring Cloud Alibaba 2023.0.3.2**
- **Nacos** - Service Registry & Configuration Center
- **Sentinel** - Flow Control & Circuit Breaking
- **Gateway** - API Gateway
- **OpenFeign** - Declarative HTTP Client
